헬스케어와 커머스 도메인에서 사용자 행동, 건강 상태, 상품 구매 패턴을 정량적으로 분석해  
전환율 개선 및 보험 전략 최적화를 주도한 데이터 분석가입니다.  
실제 의사결정과 연결되는 **분석 설계, 지표 개발, 모델링, 실험 설계**에 강점을 가지고 있으며  
Python, SQL, R 기반 분석 전 과정을 독립적으로 수행할 수 있습니다.
